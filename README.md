# Intro - timer and stopwatch
built with 100% python(and minor bash code), this project looks to be used for streamers who dont want to use a google timer or Snaz because its confusing and dumb.

# Use
when using OBS or Streamlabs add a window capture, change the window match prioritiy to __Window title must match__ and deselect capture cursor.
when closing the window hit quit on the windows or you can just close the teminal, ittle scream at you if you use quit ignore it its fine and then close the terminal.

to start the application normally open a terminal and __cd <path to stopwatch-overlay> && python main.py__
# IF YOURE STARTING IT FOR THE FIRST TIME READ THIS
run __cd \<path to install file> && ./setup.<depends on system, cmd for windows sh for linux>__
